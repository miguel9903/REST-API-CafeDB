This project is an example of a REST API, which allows to obtain, insert, edit and delete data about a cafeteria. The data that may be obtained are the following: Users, Products and Categories.

The project was carried out using NodeJS and the Express framework, technologies used to build the server. The NoSQL MongoDB database system and the MongoDB Atlas cloud database service were used for the database implementation.
